## ЛЕНИВЫЙ клиент для взаимодействия с базами данных

### В планах:

- Создать полноценный tui приложение для взаимодействия с популярными базами данных (Postgre, MySql, Oracle, ClickHouse)
- Предоставление удобного интерфейса контроля базой данных, просмотр нагруженности, заполненности, фильтрация и поиск по данным без глубокого знания sql
- Интеграция в nvim/vim
- Блэйзинг фаст код на расте

### Немного описания

Мне очень нравится lazygit, регулярно им пользуюсь, а так же я постоянно использую pgcli, он удобный, касотомизабельный, но всё ещё cli клиент.
DBeaver и прочие штуки мне очень не нравятся, они тяжёлые и крайне высокий уровень абстракции да и много функционала непонятного.

**Важно**, что это мой первый не hello world проект на расте, поэтому код местами может быть достаточно стрёмным, код свободный, если кому нравится - юзайте)
